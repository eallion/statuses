[#TIL](https://e5n.cc/tags/TIL) 大鹅 Daed 劫持域名到本地，先设置安装青龙面板的 IP， /etc/hosts：  
192.168.0.65537 qinglong.eallion.com

再配置 Daed DNS：  
routing {  
request {  
# ......  
# ......  
# ......  
  
# 劫持服务到本地  
qname(suffix: qinglong.eallion.com) -> asis  
fallback: alidns  
}  
}
