[#TIL](https://e5n.cc/tags/TIL) Hugo 官方 Docker 镜像构健命令：

docker run -it --rm \  
-v $(pwd):/project \  
-p 1313:1313 \  
ghcr.io/gohugoio/hugo:v0.145.0 \  
server -w -D -p 1313 --bind 0.0.0.0 --contentDir example -e production --enableGitInfo --minify --disableKinds RSS --renderToMemory

挂载卷是 /project
